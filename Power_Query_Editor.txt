What is Query in PQE?
Query in PQE are the tables imported or the source data imported.

UI of PQE

1. Ribbon - Which contains all the transformations/options available in PQE. (Tabs --> Groupos --> Options).
2. Data View/Result View - Displays the data.
3. Query Pane - It contains all the data source/Tables.
4. Query settings pane
	Properties - Displays the selected file.
	Applied Steps - Will display the order of steps applied on the Query.
5. Formula bar - Displays the M-language.

------------------------------------------------------
File Tab

 ***************Power bi file types******************
1. pbix - this inlcudes all the work done in PBI Desktop.
2. pbids - Power bi dataset (this inlcudes all the sources imported).
3. pbiviz - power bi visual.

Save - Save the Pbix file.
Save as - Save the Pbix file.
Apply - This will apply the changes performed in PQE to PBI Desktop.
Close - This will close PQE without appling the changes.
Close and Apply - This will apply the changes performed in PQE to PBI Desktop and close the PQE.
Options - This has all the options to apply in PowerBI.
Options and settings --> Datasource settings
1. Data sources in current file - This will display list for source file connected.
2. Global permissions - This will display the origin file.
3. Change Source - This will help to change the source file.
4. Export PBIDS - Exports the data sources together as one dataset file.
5. Edit permissions - Give the permission access at different level for the dataset/data source
	Private, public, organizational, none.
6. Clear permissions - deletes the permission on that datasource.

Home Tab

Close - Close, Apply, Close and apply.
New Source - To connect to new datasource.
Recent sources - To connect to the recently connected datasource.
Enter data - Create new data manually.
Data source settings - compelete working on datasources connected.
Refresh
	Refresh Preview - Single query refresh.
	Refresh all - This will refresh all the queries in PQE.
	Cancel Refresh - This will help to cancle the refresh in between.
Properties - It tells you the selected query.
Advanced Editor - This will display completed M - language for all the applied steps for a Query in PQE.
Delete - deletes the complete query/table in PQE.
Duplicate - Duplicates the query along with the applied steps.
Reference - This alos depliate the query but all the applied steps are together given as Source.
